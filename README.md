# CherryBlossomGrid.WPF

CherryBlossomGrid.WPF 是一个基于WPF框架的美观的Grid。


## 快速开始
```XAML
xmlns:wpf="clr-namespace:CherryBlossomGrid.WPF;assembly=CherryBlossomGrid.WPF"
```
```XAML
  <wpf:CherryBlossomAniGrid>
        <Grid>
           
        </Grid>
    </wpf:CherryBlossomAniGrid>
```
### 安装

通过 NuGet 包管理器安装：

```powershell
Install-Package CherryBlossomGrid.WPF
```
